# Disease Prediction Project

## Overview
This project applies machine learning techniques to predict the likelihood of diseases based on patient health data. It includes models for diabetes, heart disease, and lung disease, built with Python and scikit-learn.

## Dataset
- CSV files containing patient health records (e.g., glucose, blood pressure, BMI, age).
- Target variable: disease outcome (positive/negative).

## Methods
- Data preprocessing: handling missing values, scaling features, encoding categorical variables.
- Models implemented: Logistic Regression, Random Forest, Support Vector Machine (SVM).
- Evaluation metrics: Accuracy, Precision, Recall, F1-score, ROC-AUC.

## Results
- Best performing model: Random Forest with high accuracy and balanced precision/recall.
- Saved trained models (`.pkl` files) included in the repository.
